Ruby Extension for Python-Markdown
==================================

Converts \[kanji](ruby:kana) to <ruby><rb>kanji</rb><rp>(</rp><rt>kana</rt><rp>)</rp></ruby>.
Its syntax is same as http://pandoc.org/scripting.html#a-filter-for-ruby-text.
